# 首页动画设计文档

## 🎨 设计理念

打造一个高级、有质感的斗牛游戏首页，融合赌场高级会所的氛围，通过精心设计的动画效果营造沉浸式体验。

## ✨ 核心动画效果

### 1. 背景层

#### 1.1 渐变背景
- **效果**: 深色渐变 `from-slate-900 via-red-950 to-slate-900`
- **风格**: 高级、神秘、赌场风
- **作用**: 营造游戏氛围基调

#### 1.2 径向光晕
- **效果**: 中心向外的红色光晕渐变
- **技术**: `radial-gradient`
- **作用**: 增加画面深度和聚焦感

#### 1.3 漂浮扑克牌 (`FloatingCards.tsx`)
- **数量**: 15张随机扑克牌
- **动画**: 
  - 垂直飘落（从上到下）
  - 3D翻转（Y轴旋转 0° → 180° → 360°）
  - Z轴旋转（增加随机性）
  - 透明度变化（淡入淡出）
- **参数**: 
  - 持续时间: 15-25秒（随机）
  - 延迟: 0-5秒（随机）
  - 位置: 随机水平位置
- **质感**: 白色卡片 + 阴影 + 3D效果

#### 1.4 粒子效果 (`ParticleEffect.tsx`)
- **数量**: 50个金色粒子
- **动画**: 闪烁效果（透明度和缩放）
- **参数**:
  - 大小: 2-6px（随机）
  - 持续时间: 2-5秒（随机）
  - 位置: 随机分布
- **作用**: 增加画面活力和梦幻感

### 2. 顶部用户栏

#### 2.1 进场动画
- **效果**: 从顶部滑入 + 淡入
- **参数**: 
  - `translateY: -100 → 0`
  - `opacity: 0 → 1`
  - 持续时间: 0.6秒
  - 缓动: easeOut

#### 2.2 头像动画
- **效果**: 轻微摇摆
- **参数**: 
  - `rotate: 0° → 10° → -10° → 0°`
  - 持续时间: 2秒
  - 循环: 无限

#### 2.3 积分数字动画
- **效果**: 数字变化时放大 + 颜色高亮
- **参数**: 
  - `scale: 1 → 1.5 → 1`
  - 持续时间: 0.3秒
- **技术**: 使用 `key={user?.balance}` 触发重新动画

### 3. 游戏标题

#### 3.1 进场动画
- **效果**: 缩放 + 上移 + 淡入
- **参数**:
  - `scale: 0.5 → 1`
  - `y: -50 → 0`
  - `opacity: 0 → 1`
  - 持续时间: 0.8秒

#### 3.2 霓虹灯效果
- **技术**: 多层渐变文字 + 阴影
- **实现**:
  1. 底层: 模糊的渐变阴影（营造光晕）
  2. 表层: 渐变文字（金色到红色）
  3. 动画: 发光强度呼吸效果
- **渐变**: `#fbbf24 → #f59e0b → #dc2626`
- **阴影动画**: 
  - 20px → 40px → 20px 光晕
  - 持续时间: 2秒
  - 循环: 无限

#### 3.3 副标题动画
- **效果**: 延迟淡入
- **延迟**: 0.5秒

### 4. 功能按钮卡片

#### 4.1 进场动画（3D效果）
- **效果**: 从下方翻转进入
- **参数**:
  - `y: 100 → 0`
  - `rotateX: -45° → 0°`
  - `opacity: 0 → 1`
  - 持续时间: 0.6秒
  - 延迟: 0.2s / 0.4s / 0.6s（依次出现）

#### 4.2 悬停动画（3D倾斜）
- **效果**: 放大 + 3D倾斜
- **参数**:
  - `scale: 1 → 1.05`
  - `rotateY: 0° → 5°`
  - `rotateX: 0° → 5°`
  - 持续时间: 0.2秒
- **技术**: `perspective: 1000px` + `transform-style: preserve-3d`

#### 4.3 点击动画
- **效果**: 按压缩小
- **参数**: `scale: 1 → 0.95`

#### 4.4 卡片层次效果
1. **光泽层**: 渐变白色半透明层（悬停时显示）
2. **边框光晕**: 白色模糊渐变边框
3. **内容层**: 3D前移效果 `translateZ(20px)`
4. **底部阴影**: 模糊的黑色阴影

#### 4.5 图标动画
- **效果**: 垂直浮动
- **参数**:
  - `y: 0 → -10 → 0`
  - 持续时间: 2秒
  - 延迟: 与进场延迟相同
  - 循环: 无限

#### 4.6 装饰线条动画
- **效果**: 悬停时线条延伸
- **参数**:
  - `width: 0 → 80px`
  - 持续时间: 0.3秒

### 5. 底部装饰

#### 5.1 进场动画
- **效果**: 从底部滑入 + 淡入
- **参数**:
  - `y: 100 → 0`
  - `opacity: 0 → 1`
  - 延迟: 0.8秒

### 6. 对话框动画

#### 6.1 背景样式
- **效果**: 深色渐变 + 金色边框 + 毛玻璃效果
- **颜色**: `from-slate-900 to-slate-800`
- **边框**: 金色半透明 `border-yellow-500/30`

#### 6.2 输入框
- **样式**: 居中大号字体 + 字母间距
- **高度**: 加高至64px
- **效果**: 黑色半透明背景

## 🎯 质感提升细节

### 1. 色彩方案
- **主色调**: 深色系（slate-900, red-950）
- **强调色**: 金色渐变（yellow-400 → amber-700）
- **辅助色**: 蓝色（blue-500）、绿色（green-500）
- **光晕**: 半透明彩色阴影

### 2. 阴影系统
- **卡片阴影**: `shadow-2xl` + 颜色阴影
- **发光效果**: `drop-shadow` + 模糊
- **底部投影**: 渐变模糊阴影

### 3. 毛玻璃效果
- **顶栏**: `backdrop-blur-md`
- **对话框**: 深色背景 + 边框

### 4. 3D效果
- **透视**: `perspective: 1000px`
- **保持3D**: `transform-style: preserve-3d`
- **层次**: `translateZ` 创造深度

### 5. 渐变技术
- **文字渐变**: `linear-gradient` + `background-clip: text`
- **背景渐变**: 多层渐变叠加
- **阴影渐变**: 模糊渐变创造光晕

## 🚀 性能优化

1. **动画性能**: 使用 `transform` 和 `opacity`（GPU加速）
2. **粒子优化**: 限制粒子数量（50个）
3. **卡片优化**: 限制飘落卡片数量（15张）
4. **循环动画**: 使用 `repeat: Infinity` 而非 JavaScript 循环
5. **pointer-events**: 背景动画设置为 `pointer-events-none`

## 🎬 动画时间轴

```
0.0s  - 背景和粒子开始
0.0s  - 顶部栏滑入
0.2s  - 标题缩放进入
0.5s  - 副标题淡入
0.2s  - 快速开始按钮翻转进入
0.4s  - 加入游戏按钮翻转进入
0.6s  - 游戏规则按钮翻转进入
0.8s  - 底部文字淡入
持续  - 标题发光呼吸（2s循环）
持续  - 头像摇摆（2s循环）
持续  - 图标浮动（2s循环）
持续  - 扑克牌飘落（15-25s循环）
持续  - 粒子闪烁（2-5s循环）
```

## 🎨 视觉层级

1. **背景层** (z-0): 渐变 + 光晕
2. **动画层** (z-0): 扑克牌 + 粒子
3. **内容层** (z-10): 所有UI元素
4. **悬浮层** (z-50): 对话框

## 💡 使用建议

1. **调整动画速度**: 修改 `duration` 参数
2. **修改颜色**: 更改渐变色值
3. **调整粒子**: 修改 `FloatingCards.tsx` 和 `ParticleEffect.tsx` 中的数量
4. **关闭动画**: 设置 `prefers-reduced-motion` 媒体查询

## 🔧 技术栈

- **Motion**: Framer Motion（motion/react）
- **样式**: Tailwind CSS 4.0
- **3D**: CSS Transform 3D
- **渐变**: CSS Gradient + Background Clip

这个设计将高级赌场的氛围与现代Web动画技术完美结合，创造出既华丽又流畅的用户体验！🎴✨
